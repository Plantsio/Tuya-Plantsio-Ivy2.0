TuyaOS
TuyaOS 简介

基于 RTOS、Linux、Non-OS 等设计的应用于 IoT 领域面向全连接、全场景的分布式跨平台操作系统。通过标准化的 Kernel 设计，解决了不同的芯片平台、系统及连接协议等带来的碎片化开发体验。TuyaOS 真正实现一次开发、多端可用，安全稳定地加快产品上市。详细介绍请查看 TuyaOS。
