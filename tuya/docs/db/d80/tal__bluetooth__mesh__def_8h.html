enum MESH_NETWORK_STATE_SET_T
枚举值
MESH_NETWORK_RESET 

Kick out, mesh node will be unprovision state, and it will clear the mesh provision data in ram and flash

MESH_NETWORK_RESET_WITH_RECOVER 

Node reset in tam, mesh node will be unprovision state, and it will clear the mesh provision data in ram. The provision data still store in flash

MESH_NETWORK_RECOVER 

Revcover the network, mesh node will be provision state, it will restore the provision data from flash into ram
